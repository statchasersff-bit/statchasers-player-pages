🔴 Efficiency Score Review

This is where we need to be careful.

Because efficiency metrics are noisy.

⚠️ QB Efficiency

You’re using:

Completion %

FP per usage

TD rate

That’s strong.

But beware:

FP per usage already includes TD effect.

TD rate separately also adds signal.

There is mild double-counting.

Not catastrophic, but something to monitor.

⚠️ RB Efficiency
yardsPerTouch − 4.0
tdPerTouch − 3

Make sure these baselines are:

League averages for RB1 tier, not overall RB pool.

If baseline is wrong, efficiency label becomes biased.

⚠️ WR/TE Efficiency

Yards per catch baseline = 8

That’s very low for WR.

League WR average is closer to 11–12.

8 is more TE territory.

If you keep 8:

Almost every WR becomes efficiency-positive.

This baseline must be calibrated per position.

This is important.

Extreme clustering

Normalize both sub-scores to 0–100 via percentile scaling.

But for now, additive is fine.

Just monitor distribution.
