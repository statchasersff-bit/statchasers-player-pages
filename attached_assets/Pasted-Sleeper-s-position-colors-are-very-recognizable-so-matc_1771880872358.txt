Sleeper’s position colors are very recognizable, so matching them will immediately feel familiar to fantasy users.

Here are Sleeper’s standard position colors (approximate hex values used across their UI):

QB → Red → #F44336

RB → Green → #4CAF50

WR → Blue → #2196F3

TE → Orange → #FF9800

K → Purple → #9C27B0

DEF → Dark Blue / Navy → #1E3A8A (Sleeper uses a deep navy tone)

Right now your pills are more pastel. If you want to match Sleeper more closely while keeping your premium StatChasers look, I recommend:

Option A — Exact Sleeper-Style (High Contrast)
Strong solid background with white text.

.sc-pos-pill {
  font-weight: 700;
  font-size: 12px;
  padding: 4px 8px;
  border-radius: 999px;
  color: #fff;
}

/* QB */
.sc-pos-qb { background: #F44336; }

/* RB */
.sc-pos-rb { background: #4CAF50; }

/* WR */
.sc-pos-wr { background: #2196F3; }

/* TE */
.sc-pos-te { background: #FF9800; }

/* K */
.sc-pos-k { background: #9C27B0; }

/* DEF */
.sc-pos-def { background: #1E3A8A; }

Option B — “Premium Sleeper” (Slightly Softer for StatChasers Brand)
Keeps Sleeper colors but tones them slightly for better visual balance with your blue/gold palette.

.sc-pos-qb { background: #E53935; }
.sc-pos-rb { background: #43A047; }
.sc-pos-wr { background: #1E88E5; }
.sc-pos-te { background: #FB8C00; }
.sc-pos-k  { background: #8E24AA; }
.sc-pos-def { background: #1E3A8A; }

If you want them even more premium, add subtle depth:

.sc-pos-pill {
  box-shadow: 0 1px 2px rgba(0,0,0,0.15);
  letter-spacing: 0.3px;
}

Important UX Note for Your Site

Because your entire ecosystem (Player Pages, Waiver Tool, Optimizer, etc.) is growing, I’d strongly recommend:

Create global CSS tokens like:

:root {
  --pos-qb: #F44336;
  --pos-rb: #4CAF50;
  --pos-wr: #2196F3;
  --pos-te: #FF9800;
  --pos-k:  #9C27B0;
  --pos-def: #1E3A8A;
}

Then reference them everywhere:

.sc-pos-qb { background: var(--pos-qb); }

This keeps all tools visually consistent across StatChasers.

Strategic Thought for You

Matching Sleeper is smart because:

Your users already associate those colors with positions.

It reduces cognitive load.

It makes your tool feel “native” to fantasy players instantly.

But keep everything else distinctly StatChasers (layout, gold accents, typography). That’s how you build brand authority instead of looking derivative.
